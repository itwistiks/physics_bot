[Главное меню]
├─ 🔢 Практика → [Тип: Случайно/Вариант/Темы]
│   └─ #Тема → Задача (условие + ответ в 1 сообщении)
├─ 📊 Статистика → Попап с данными
├─ 👨‍🏫 Репетитор → Ссылка на сайт
└─ ✉️ Поддержка → Форма обратной связи

Команды:
/stop - прервать задание
/theory - быстрый доступ

---------------------------------------------------------------------------------------------------------
Обозначения
---------------------------------------------------------------------------------------------------------

Обозначения для телеграм бота:
	[Текст кнопки] - Меню, кнопки что появляются вместо клавиатуры
	(Текст/картинка) + *Текст кнопки* - Сообщение + кнопки что появляются под сообщением
	<Текст> - Выполнение какого либо действия
	/Текст/ - Дополнительные пояснение

---------------------------------------------------------------------------------------------------------
Фронтенд
---------------------------------------------------------------------------------------------------------

(Текст бота) → </start> → (Сообщение приветствия) + <Открывается основное меню>

Основное меню:
	[Практика] → <Открывается меню практикума>
		Меню практикума:
			[Задания] → <Открывается меню заданий>
				Меню заданий:
					[Случайно] → <Случайные задачи из бд и выполняется Задача сообщением>
					[Первая часть] → <Выводит вместо сообщении кнопки с номерами первой части> /Например: *1*+*2*+*3*...*14*+*15*+*16*/ → <Стирается сообщение и в зависимости от номера выполняется Задача сообщением>
					[Вторая часть] → <Выводит вместо сообщения кнопки с номерами второй части> /Например: *17*+*18*+*19*+*20*+*21*+*22*/ → <Стирается сообщение и в зависимости от номера выполняется Задача сообщением>
					[Назад] → <Возвращает в Меню практикума>
			[Вариант] → <Составляется случайный вариант по типам заданий и выполняется Задача сообщением>
			[Темы] → *Механические явления* + *Тепловые явления* + *Электрические явления* + *Квантовые явления* → <Стирается сообщение и в зависимости от темы выводятся *Кинематика* + *Динамика* + *Статика* или *МКТ* + *Термодинамика* или *Электростатика* + *Электрический ток* + *Магнетизм* или *Оптика* + *Реакции*> → <Стирается сообщение и выполняется Задача сообщением>
			[Сложные задачи] → *Механические явления* + *Тепловые явления* + *Электрические явления* + *Квантовые явления* → <Стирается сообщение и выполняется Задача сообщением>
			[Буст оценки] → <Происходит тоже самое, что и при нажатии на Забустить оценку>
			[Назад] → <Возвращает в основное меню>

	[Статистика] → *Место в топе недели* + *Общее место в топе* + *Твой рекорд дня* + *Всего правильных* + *Твоя слабая тема* + *Текущий уровень* + *Достижения* → <Выводит нужную инфомацию>
	[Улучшить знания] → *Купиить подписку* + *Заниматься с преподователем* → <Переход на сайт для записи>
	[Другие предметы] → (В разработке) <Стирается сообщение через несколько секунд>
	[Письмо в поддержку] → (Напишите свои пожелания) + *Отменить* → (Сообщение отправлено) <В случае отмены стирается сообщение>
	[Отчистить чат] → <Удаляет все сообщения>



Задача сообщением: /Выбираются задачи из базы данных MySQL по конкретному запросу, формируя список id задач и поочередно формируется в такие сообщения/
	(Условие задачи) + (Картинка к задаче, если есть, распологается над условием) +
	(Опрос с вариантами ответов) /Инлайн-кнопки для ответа; Кнопки *Теория* + *Разбор* + *Видео разбор (если есть)* (появляются после ответа); / /Добавить эффекты на правильный ответ и не правильный/
	[Следующее задание] → <Выводит следующую Задачу сообщением> /Задание отправляется в конец списка/
	[Остановиться] → <Возвращает в Меню практикума>
	


Пример сообщения:
📝 Задача 14:
"Тело движется... (рисунок)"

Варианты:
[A] 10 м/с  [B] 15 м/с  [C] 20 м/с  [D] 25 м/с

Дополнительно:
[Теория]  [Пример решения]
