(2295, 3, 7, 'BASIC', '<strong>Закон Ома для участка цепи и последовательное соединение</strong>

Яркость свечения лампочки в цепи постоянного тока определяется выделяющейся на ней мощностью. Мощность вычисляется по формуле: P = I² * R, где I - сила тока в цепи, R - сопротивление лампочки.

При <i>последовательном</i> соединении проводников:
- Сила тока одинакова на всех участках цепи: I = I₁ = I₂
- Общее сопротивление цепи равно сумме сопротивлений: R_общ = R₁ + R₂
- Общее напряжение равно сумме напряжений на отдельных участках: U_общ = U₁ + U₂

Сила тока в цепи по закону Ома: I = U / R_общ, где U - ЭДС источника.

Чтобы лампочка горела ярче, нужно <i>увеличить</i> силу тока в цепи, так как мощность пропорциональна квадрату силы тока (P = I² * R_лам). Сила тока увеличится, если <i>уменьшить</i> общее сопротивление цепи R_общ.

<i>Анализ вариантов:</i>
1) Добавить такую же проволоку последовательно -> R_общ увеличивается (R_общ = R_пр + R_пр + R_лам) -> I уменьшается -> яркость падает.
2) Заменить железо на нихром -> У нихрома удельное сопротивление больше -> R_пр увеличивается -> R_общ увеличивается -> I уменьшается -> яркость падает.
3) Поменять местами -> При последовательном соединении порядок элементов не важен. R_общ не меняется -> I не меняется -> яркость не меняется.
4) Добавить такую же проволоку параллельно -> Общее сопротивление участка с двумя параллельными проволоками станет меньше, чем сопротивление одной (R_паралл = (R_пр * R_пр) / (R_пр + R_пр) = R_пр/2) -> R_общ цепи уменьшится -> I увеличится -> яркость увеличится.
